# Updates

## 2026-01-17
- Added per-type auto numbering for newly created elements (Rectangle 1, Text 2, etc.).
- Select tool no longer selects frame bodies; frame selection is via label or Layers panel.
- Ctrl/Cmd + D duplicates selected or last-created element with +40px X offset.
- Added align controls with Align-to Selection/Frame toggle in Properties.
- SVG imports now expose editable vector nodes; path editing supports vertex handles and segment insertion.
- Pen tool editing: vertex handles, hover feedback, segment highlight, and last segment visibility.
- Hover highlight on selectable elements in Select tool.

## 2026-01-18
- Consolidated playback into Play-only mode and removed Preview UI.
- Play now centers the starting frame and ignores edit pan/zoom while active.
- Added Flow starting frame selection (Animate tab) with canvas badge.
